HTML - Stands for Hypertext Markup Language. HTML provides the basic structure for a website. It uses in-line tags to define how the website is structured. Tags define "elements", which combined make the website.

CSS- stands for Cascading Style Sheets. It is used to style markup languages. It isn't useful on its own, you have to use it on connection with something like a HTML document. CSS will bring the HTML to life by allowing you to change fonts, apply colors, define margins and more.

Javascript - Javascript is an interpreted scripting language that defines the behaviour of the website. JS can be used to script interactions, access databases and add general interactivity to yoru site.
